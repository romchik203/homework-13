# homework-13

## Цель работы

Написать Python-скрипт, который обращается к API антивирусной системы VirusTotal и выводит результат в формате JSON, а также формирует краткий читаемый отчёт.

## Выполненная работа

Реализован скрипт `vt_file_report.py`, который:

1. При запуске запрашивает API-ключ VirusTotal (ключ не хранится в коде).
2. Запрашивает хеш файла (MD5, SHA-1 или SHA-256) — можно ввести с клавиатуры или передать аргументом командной строки. Если хеш не указан, используется тестовый хеш EICAR.
3. Выполняет GET-запрос к VirusTotal API v3 (`/files/{hash}`).
4. Выводит полный ответ в консоль в формате JSON.
5. Сохраняет сырой ответ в файл `virustotal_response.json`.
6. Парсит ответ и сохраняет краткий отчёт в файл `virustotal_report.txt` (информация о файле, хеши, статистика по движкам, список антивирусов, обнаруживших угрозу, теги, вердикты песочниц, дата анализа).

Использованы стандартная библиотека (json, sys, pathlib, datetime) и библиотека `requests` для HTTP-запросов.

## Требования

- Python 3.6+
- Установка зависимостей: `pip install requests`

## Как запускать

1. Получить бесплатный API-ключ на https://www.virustotal.com (раздел API Key в профиле).
2. Выполнить: `python vt_file_report.py`
3. По запросу ввести API-ключ и хеш файла (или нажать Enter для тестового хеша).
